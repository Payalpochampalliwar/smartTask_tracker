## How to Run Locally

### 1. Clone the Repository
git clone https://github.com/Payalpochampalliwar/smartTask_tracker.git
cd smart-task-tracker

### 2. Install Dependencies
npm install

### 3. Start the App

npm start
App will open at http://localhost:3000/
